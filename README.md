# colorful-reversi
JavaScriptで作る変則リバーシ

https://y-tetsu.github.io/colorful-reversi/

<img src="image/demo1.gif" width="48%">

盤面の形や色、石の初期配置が異なる変則リバーシです。<br>
黒が人、白はコンピュータ(ランダム)です。<br>

## 実行方法
ソース一式をダウンロード後、index.htmlを任意のブラウザで開く。<br>
