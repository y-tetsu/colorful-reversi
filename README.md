黒と白に加え、灰色の3人対戦のリバーシです。<br>
盤面形状、初期配置、盤面の色も通常から変更しています。<br>

黒が人、白と灰色はコンピュータです。<br>
灰色はランダムで、白はやや強いです。<br>

## 実行方法
ソース一式をダウンロード後、index.htmlを任意のブラウザで開く。<br>
